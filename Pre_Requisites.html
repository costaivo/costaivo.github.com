Prerequisites

  1. One of the supported operating systems (MacOS, Windows 11, Linux)
  2. Java (JDK & JVM) or can even use any of the open source JDK (https://corretto.aws/downloads/latest/amazon-corretto-8-x64-windows-jdk.msi)
  3. Chrome browser for browser based automation
  4. Any text editor of your choice for creation of workflows (Ideally VSCode)

Installation

Visit Tagui installation page for guided installation.
https://tagui.readthedocs.io/en/latest/setup.html

  1. Download https://github.com/kelaberetiv/TagUI/releases/download/v6.110.0/TagUI_Windows.zip
  2. Visit the location of the downloaded file and extract the zip to the desired location.
    Ex. D:/Projects/Tagui
  3. Install OpenJDK or JDK 17. (https://corretto.aws/downloads/latest/amazon-corretto-8-x64-windows-jdk.msi)
  4. Install latest version of chrome browser if not already installed on your system.
    (https://www.google.com/chrome/what-you-make-of-it/)
  5. Click on the start menu, search for Settings > System > About > Advanced System Settings > Environment Variables
  6. Add path D:/Projects/Tagui/** to the System Environment Variables
  7. Once complete, open command prompt and type the following command:
    tagui
  8. If you get "php.exe - System Error" then visit the trouble shooting page and install Visual C++ Redistributable.
    (https://tagui.readthedocs.io/en/latest/setup_troubleshooting.html#troubleshooting-windows)
  9. Now run the command from Step 7 again and if there were no errors then you've successfully installed tagui
  10. Verify the installation by simply running one of the sample workflows by firing the following command:
    tagui D:/Projects/Tagui/flows/sample/1_google.tag
  11. If the automation is able to complete by opening up a chrome instance, then you've completed the installation process successfully.
