Session: Automation

Have you ever wondered what it be like to clone yourself. That is someone sitting at your desk in your place, doing your job for you while you sip on a cup of coffee? It would be something else right? Well, this is exactly what is called as having a "bot". Bot is a program which is a collection of tasks, rules, keyboard and mouse interactions. It typically imitates human interactions with different applications.

  1. Attended: Programmed to work along with human intervention
  2. Unattended: Fully automated by teaching robot how to perform tasks

RPA is a software program that imitates human actions while interacting with a computer application and accomplishing automation of repetitive rule-based tasks. It makes use of bots to perform the actions.

  • Data entry
  • Copy/paste
  • Multiple app interactions
  • Authentication
  • Verification Rule/Check
  • Navigation

  • Automation Anywhere
  • UiPath
  • TagUI

  1. TagUi is open source
  2. It is cross-platform, i.e., it supports Windows, macOS, and Linux.
  3. Supports visual automation of web and desktop applications using SikuliX. SikuliX is a computer vision library that allows image-based identification of objects on the screen.
  4. Supports writing code in 20+ human-spoken languages. Additionally, it supports Python and JavaScript.
  5. Smaller learning curve compared to other software tools

  • Installation guide

It comprises of 4 key components:

  1. Flow file: .tag extension file which contains all your code
  2. Flow: is a collection/series of steps
  3. Step: is basically a rule or an action that needs to be performed
  4. Identifier: It is used to locate different parts of the screen

Identifier will by default try to match id, name or classname of the elements present in the DOM.

Click Search

This will click on any element that has either id, name or class name as "Search".

>>Absolute identifier XPath:

/html/body/div[1]/div[3]input

>>Relative identifier XPath:

//*[@id="search-path"]/input

When it is difficult to locate any element using the default, absolute or relative identifiers, we can use visual identifiers to locate the element. This can be done by simply capturing a snapshot of the element and then referencing that image in the action as follows:

click elements/search.png

Read //span[@class="stock-name"] to stockName
echo stockName

Sometimes you may realize that your computer is faster to process your automation, but your network is too slow to respond. It could very likely be that you are having connectivity issues with your ISP. In such cases, we need to synchronize the computer with your network responses. To do this, we can use two methods:

  1. Wait: Explicitly tells the program to wait for a specific number of seconds before moving on to the next line
  2. Timeout: Tells the program to wait at most x number of seconds before terminating. If the page loads before that, then the program terminates immediately.
wait 5
wait 5 seconds
timeout 30

There are four types of click events:

  1. click: Performs a left click
  2. dclick: Performs a double click
  3. rclick: Performs a right click
  4. hover: Performs a hover over an element

Keyboard actions are of two types:

  1. keyboard: This will send keystrokes like how a user would type on the keyboard
  2. type: Send full keyed texts to specific elements in the web interface
click input
keyboard Hello World
keyboard [enter]
keyboard [win]r
keyboard [space]

type input as Hello World
type input as [clear]Hello Learner[enter]
type searchbox.png as Tomatoes[enter]

To display the URL of the page you are currently on, you can use the url() helper function.

currentURL = `url()`
echo `currentURL`

if currentURL equal to "https://example.com/success"
echo "Transaction was successful!"

To display the title of the page you are currently on, you can use the title() helper function.

https://www.bankrate.com
timeout 5

echo `title()`

To get the contents of the current page you are currently on, you can use the text() helper function.

https://www.investopedia.com
echo `text()`

if text() contains 'Investing'
echo 'Investing option is displayed!'

To display the amount of time elapsed between executions, you can use the timer() helper function.

timer()

http://www.google.com
timeout 5

echo total time = `timer()`

To copy and paste content onto the clipboard, you can use the clipboard() helper function.

varGetContent = ""

echo Before clipboard varGetContent: `varGetContent`

clipboard('tagui text in clipboard')

varGetContent = clipboard()

echo After clipboard varGetContent: `varGetContent`

To get the count of the number of matching elements matching the specified identifier, you can use the count() helper function.

varCountLink = 0

echo varCountLink = `varCountLink`

https://www.bankrate.com
timeout 5

varCountLink - count('Credit cards')

echo varCountLink = `varCountLink`

To check if an element exists on the screen, you can use the exist() helper function. This waits for the timeout. To check if an element exists on the screen without waiting for the timeout, you can use the present() helper function.

varResult = ""

https://www.bankrate.com
timer()

varResult = exist('Credit cards')
echo varResult - `varResult`
echo time taken = `timer()`

varResult = present('Credit cards')
echo varResult - `varResult`
echo time taken = `timer()`

To get the environment variables from the system, you can use the get_env() helper function.

echo TEMP: `get_env('TEMP')`
echo OS: `get_env('OS')`
echo USERNAME: `get_env('USERNAME')`

To capture a screenshot of the screen, you can use the snap() helper function.

// snap [DOM/XPath/Region/Page] to [filename]

https://www.investopedia.com

snap Education to education.png

snap //a[@id='header_logo'] to invest_logo.png

To save texts to a new file, you can use the dump() helper function.

// This command will always overwrite a file, if the file with the same name already exists.
// dump [text] to [filename]


https://www.investopedia.com
dump text() to text_file.txt

To append to an existing file, you can use the write() helper function.

write Add this line of text to file1.txt
write Add this line of text again to file1.txt

To load the contents of a file into a variable, you can use the load() helper function.

fileContents = load(file1.txt)
echo `fileContents`
